## Installation

1.  Launch Visual Studio Code
2.  Open Extensions sidebar panel in VS Code
3.  Search for `highflyer theme`
4.  Click **Install** to install it
5. From the menu bar click: Code > Preferences > Color Theme > `Highflyer theme`
